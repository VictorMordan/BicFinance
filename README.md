# 📈 BicFinance - Control de Gastos

**BicFinance** es una aplicación web y PWA (Progressive Web App) diseñada para facilitar la gestión y el control de tus finanzas personales de forma rápida y sencilla. La aplicación te permite establecer un presupuesto del mes, monitorear tus balances y registrar tus gastos en tiempo real con almacenamiento seguro en la nube.

---

## 🛠️ Tecnologías Utilizadas

*   **Frontend**: HTML5, Tailwind CSS y JavaScript Moderno.
*   **Base de Datos y Seguridad**: Firebase (Firestore Database y Authentication con Google).
*   **Características PWA**: Service Worker para soporte básico fuera de línea y archivo de manifiesto para instalación en el celular.
